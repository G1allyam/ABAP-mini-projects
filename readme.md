# ABAP Mini Projects

## 1. Hello SAP World
Simple ABAP report that prints:
- current user
- system date
- system time

### Technologies:
- ABAP Report
- System fields (SY-UNAME, SY-DATUM, SY-UZEIT)

## 2. Customer List

ABAP report that demonstrates working with internal tables.
The program creates a list of customers and displays it in a simple report format.

Features:

Define custom structures using TYPES

Use internal tables

Loop through data and display results

### Technologies:

ABAP Report

Internal Tables

LOOP / APPEND

## 3. Sales Report

ABAP report that simulates a basic sales report with calculated totals.
The program calculates total sales per product and a grand total.

Features:

Business logic calculations

Computed fields

Summary totals

### Technologies:

ABAP Report

Arithmetic operations

Data aggregation

## 4. String Operations

ABAP program demonstrating common string operations used in real projects.

Features:

Convert text to uppercase

Calculate string length

Extract substrings

### Technologies:

ABAP String Functions

STRING, STRLEN, TO_UPPER

## 5. Internal Table CRUD

ABAP report demonstrating CRUD (Create, Read, Update, Delete) operations
on an internal table, similar to real-life SAP data handling.

Features:

Create records

Read records using keys

Update existing entries

Delete records

### Technologies:

ABAP Report

Internal Tables

READ TABLE, MODIFY, DELETE

SY-SUBRC

## Project Purpose

These mini projects were created to:

Practice core ABAP concepts

Demonstrate clean and readable ABAP code

Serve as a beginner-friendly ABAP portfolio
